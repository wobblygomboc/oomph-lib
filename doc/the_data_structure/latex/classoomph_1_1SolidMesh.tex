\doxysubsection{Detailed Description}
General \mbox{\hyperlink{classoomph_1_1SolidMesh}{Solid\+Mesh}} class. 

Solid meshes contain Solid\+Finite\+Elements which contain Solid\+Nodes. This class simply overloads the appropriate access functions from the underlying \mbox{\hyperlink{classoomph_1_1Mesh}{Mesh}} class.
