class oomph\+::\+FSIDriven\+Cavity\+Mesh$<$ ELEMENT $>$}
\mbox{\hyperlink{classoomph_1_1Mesh}{Mesh}} for W. Wall\textquotesingle{}s FSI driven cavity problem. The mesh is derived from the {\ttfamily \mbox{\hyperlink{classoomph_1_1SimpleRectangularQuadMesh}{Simple\+Rectangular\+Quad\+Mesh}}} so it\textquotesingle{}s node and element numbering scheme is the same as in that mesh. Only the boundaries are numbered differently to allow the easy identification of the \char`\"{}collapsible\char`\"{} segment. Boundary coordinates are set up for all nodes located on boundary 3 (the collapsible segment). The curvilinear (\char`\"{}collapsible\char`\"{}) segment is defined by a {\ttfamily \mbox{\hyperlink{classoomph_1_1GeomObject}{Geom\+Object}}}. 


\begin{DoxyItemize}
\item Boundary 0 is the moving lid.
\item Boundary 1 is the gap above the moving lid on the right wall
\item Boundary 2 is the rigid part of the right wall
\item Boundary 3 is the moving (elastic) wall
\item Boundary 4 is the rigid part of the left wall
\item Boundary 5 is the gap above the moving lid on the left wall 
\end{DoxyItemize}
