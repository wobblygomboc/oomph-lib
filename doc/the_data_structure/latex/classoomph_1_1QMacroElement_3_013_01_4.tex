\doxysubsection{Detailed Description}
\mbox{\hyperlink{classoomph_1_1QMacroElement}{QMacro\+Element}} specialised to 3 spatial dimensions. 

The macro element mapping is based on the member function pointer to the associated \mbox{\hyperlink{classoomph_1_1Domain}{Domain}} \textquotesingle{}s 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classoomph_1_1Domain_a95f3e00d28ea37e6c4d3027bfac91096}{Domain::macro\_element\_boundary}}(...) }

\end{DoxyCode}
 function which provides a parametrisation of the macro element\textquotesingle{}s boundaries in the form\+: \[ {\bf f}_{i} (t,{\bf S}) \] where
\begin{DoxyItemize}
\item $ i $ labels the boundary (L/\+R/\+D/\+U/\+B/F)
\item $ {\bf S} \in [-1,1]^2 $ is the (2D) \mbox{\hyperlink{classoomph_1_1Vector}{Vector}} of local coordinate(s) along the boundary.
\item $ {\bf f} $ is the position \mbox{\hyperlink{classoomph_1_1Vector}{Vector}} to the boundary.
\item $ t $ is the time level (t=0\+: current; t\texorpdfstring{$>$}{>}0 previous timestep) 
\end{DoxyItemize}
