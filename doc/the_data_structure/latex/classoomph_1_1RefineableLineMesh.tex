class oomph\+::\+Refineable\+Line\+Mesh$<$ ELEMENT $>$}
Intermediate mesh class that implements the mesh adaptation functions specified in the Refineable\+Mesh class for meshes that contain the refineable variant of \mbox{\hyperlink{classoomph_1_1QElement}{QElement}} s \mbox{[}The class ELEMENT provided as the template parameter must be of type \mbox{\hyperlink{classoomph_1_1RefineableQElement_3_011_01_4}{Refineable\+QElement$<$1$>$}}\mbox{]}. 

\mbox{\hyperlink{classoomph_1_1Mesh}{Mesh}} adaptation/refinement is implemented by \mbox{\hyperlink{classoomph_1_1BinaryTree}{Binary\+Tree}} procedures and any concrete implementation of this class needs to provide a \mbox{\hyperlink{classoomph_1_1BinaryTreeForest}{Binary\+Tree\+Forest}} representation of the initial (coarse) mesh.
