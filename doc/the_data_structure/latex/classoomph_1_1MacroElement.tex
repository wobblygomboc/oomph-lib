\doxysubsection{Detailed Description}
Base class for \mbox{\hyperlink{classoomph_1_1MacroElement}{Macro\+Element}} s that are used during mesh refinement in domains with curvlinear and/or time-\/dependent boundaries; see the description of the \mbox{\hyperlink{classoomph_1_1Domain}{Domain}} class for more details.
