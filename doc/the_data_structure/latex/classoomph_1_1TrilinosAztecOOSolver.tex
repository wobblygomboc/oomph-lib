An interface to the Trilinos Aztec\+OO classes allowing it to be used as an Oomph-\/lib \mbox{\hyperlink{classoomph_1_1LinearSolver}{Linear\+Solver}}. The Aztec\+OO solver is a Krylov Subspace solver; the solver type (either \mbox{\hyperlink{classoomph_1_1CG}{CG}}, \mbox{\hyperlink{classoomph_1_1GMRES}{GMRES}} or \mbox{\hyperlink{classoomph_1_1BiCGStab}{Bi\+CGStab}}) can be set using \mbox{\hyperlink{classoomph_1_1TrilinosAztecOOSolver_a7c81cd1f21720d2c05db9d499dc8f2bc}{solver\+\_\+type()}}. This solver can be preconditioned with Trilinos Preconditioners (derived from \mbox{\hyperlink{classoomph_1_1TrilinosPreconditionerBase}{Trilinos\+Preconditioner\+Base}}) or Oomph-\/lib preconditioners (derived from \mbox{\hyperlink{classoomph_1_1Preconditioner}{Preconditioner}}). Preconditioners are set using \mbox{\hyperlink{classoomph_1_1IterativeLinearSolver_ad4ae617ce8304f17bd42f4dc39813e22}{preconditioner\+\_\+pt()}}.

\doxysubsection{Detailed Description}
An interface to the Trilinos Aztec\+OO classes allowing it to be used as an Oomph-\/lib \mbox{\hyperlink{classoomph_1_1LinearSolver}{Linear\+Solver}}. The Aztec\+OO solver is a Krylov Subspace solver; the solver type (either \mbox{\hyperlink{classoomph_1_1CG}{CG}}, \mbox{\hyperlink{classoomph_1_1GMRES}{GMRES}} or \mbox{\hyperlink{classoomph_1_1BiCGStab}{Bi\+CGStab}}) can be set using \mbox{\hyperlink{classoomph_1_1TrilinosAztecOOSolver_a7c81cd1f21720d2c05db9d499dc8f2bc}{solver\+\_\+type()}}. This solver can be preconditioned with Trilinos Preconditioners (derived from \mbox{\hyperlink{classoomph_1_1TrilinosPreconditionerBase}{Trilinos\+Preconditioner\+Base}}) or Oomph-\/lib preconditioners (derived from \mbox{\hyperlink{classoomph_1_1Preconditioner}{Preconditioner}}). Preconditioners are set using \mbox{\hyperlink{classoomph_1_1IterativeLinearSolver_ad4ae617ce8304f17bd42f4dc39813e22}{preconditioner\+\_\+pt()}}.
