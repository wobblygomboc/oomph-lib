\hypertarget{classoomph_1_1Data}{}\doxysection{oomph\+::Data Class Reference}
\label{classoomph_1_1Data}\index{oomph::Data@{oomph::Data}}


A class that represents a collection of data; each \mbox{\hyperlink{classoomph_1_1Data}{Data}} object may contain many different individual values, as would be natural in non-\/scalar problems. \mbox{\hyperlink{classoomph_1_1Data}{Data}} provides storage for auxiliary `history' values that are used by \mbox{\hyperlink{classoomph_1_1TimeStepper}{Time\+Stepper}} objects to calculate the time derivatives of the stored data and also stores a pointer to the appropriate \mbox{\hyperlink{classoomph_1_1TimeStepper}{Time\+Stepper}} object. In addition, an associated (global) equation number is stored for each value.

\doxysubsection{Detailed Description}
A class that represents a collection of data; each \mbox{\hyperlink{classoomph_1_1Data}{Data}} object may contain many different individual values, as would be natural in non-\/scalar problems. \mbox{\hyperlink{classoomph_1_1Data}{Data}} provides storage for auxiliary `history' values that are used by \mbox{\hyperlink{classoomph_1_1TimeStepper}{Time\+Stepper}} objects to calculate the time derivatives of the stored data and also stores a pointer to the appropriate \mbox{\hyperlink{classoomph_1_1TimeStepper}{Time\+Stepper}} object. In addition, an associated (global) equation number is stored for each value. 

The \mbox{\hyperlink{classoomph_1_1Data}{Data}} class permits copies of the stored data values and equation numbers into another \mbox{\hyperlink{classoomph_1_1Data}{Data}} object using the \mbox{\hyperlink{classoomph_1_1Data_a57889a9db7ca009434f0268173fcf3f8}{copy()}} function. Shallow (pointer based) copies of the values can be obtained by using specific derived classes. In such cases pointers to the objects that contain the pointer-\/based copies should be stored in the original \mbox{\hyperlink{classoomph_1_1Data}{Data}} class (in the array Copy\+\_\+of\+\_\+data\+\_\+pt) so that resize and destruction operations can be performed safely.
