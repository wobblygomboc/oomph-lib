class oomph\+::\+Face\+Element\+As\+Geom\+Object$<$ ELEMENT $>$}
Class that is used to create \mbox{\hyperlink{classoomph_1_1FaceElement}{Face\+Element}} from bulk elements and to provide these \mbox{\hyperlink{classoomph_1_1FaceElement}{Face\+Element}} with a geometric object representation. The local coordinates of the Face\+Elements are used as the intrinisic coordinates for its \mbox{\hyperlink{classoomph_1_1GeomObject}{Geom\+Object}} representation. 

These elements are used primarily to set up the interaction terms in FSI problems and are expected to be created from meshes so that they lie on a particular boundary of the mesh.
