/////////////////////////////////////////////////////////////////////////// /////////////////////////////////////////////////////////////////////////// 

An Element that solves the equations of solid mechanics, using the principle of virtual displacements, with quadratic interpolation for the positions and a discontinuous linear solid pressure. This is analogous to the \mbox{\hyperlink{classoomph_1_1QCrouzeixRaviartElement}{QCrouzeix\+Raviart\+Element}} element for fluids.
