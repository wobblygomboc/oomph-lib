The following equation is implemented \+: du/ds\+\_\+n = dt/ds\+\_\+n $\ast$ ds\+\_\+t/dt $\ast$ du/dt. The bulk biharmonic elements on the boundary must be hijackable and the du/ds\+\_\+n and d2u/ds\+\_\+nds\+\_\+t boundary DOFs hijacked when these elements are applied.
