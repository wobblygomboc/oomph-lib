\doxysubsection{Detailed Description}
\subsubsection*{template$<$class ELEMENT$>$\newline
class oomph\+::\+Spine\+Surface\+Volume\+Constraint\+Bounding\+Element$<$ ELEMENT $>$}
////////////////////////////////////////////////////////////////////// ////////////////////////////////////////////////////////////////////// 

The Two-\/dimensional interface elements that allow the application of a volume constraint specialised for the case when the nodal positions of the bulk elements are adjusted using spines. To enforce that a fluid volume has a certain volume, attach these elements to all faces of the (3D Cartesian) bulk fluid elements (of type ELEMENT) that bound that region and then specify the \char`\"{}pressure\char`\"{} value that is traded for the constraint.
