A template Class for Boundary\+Nodes; that is Nodes that MAY live on the boundary of a \mbox{\hyperlink{classoomph_1_1Mesh}{Mesh}}. The class is formed by a simple composition of the template parameter NODE\+\_\+\+TYPE, which must be a Node class and the \mbox{\hyperlink{classoomph_1_1BoundaryNodeBase}{Boundary\+Node\+Base}} class. Final overloading of functions is always in favour of the \mbox{\hyperlink{classoomph_1_1BoundaryNodeBase}{Boundary\+Node\+Base}} implementation; i.\+e. these nodes can live on boundaries.
