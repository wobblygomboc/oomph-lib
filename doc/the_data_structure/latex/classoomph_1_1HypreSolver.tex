\doxysubsection{Detailed Description}
///////////////////////////////////////////////////////////////////// ///////////////////////////////////////////////////////////////////// 

An \mbox{\hyperlink{classoomph_1_1LinearSolver}{Linear\+Solver}} class using the suite of Hypre solvers to allow

Boomer\+AMG (AMG), \mbox{\hyperlink{classoomph_1_1CG}{CG}}, \mbox{\hyperlink{classoomph_1_1GMRES}{GMRES}} or \mbox{\hyperlink{classoomph_1_1BiCGStab}{Bi\+CGStab}}

to be used for Linear\+Solver\+::solve(...) or Linear\+Solver\+::resolve(...).

The Krylov subspace solvers (\mbox{\hyperlink{classoomph_1_1CG}{CG}}, \mbox{\hyperlink{classoomph_1_1GMRES}{GMRES}} and \mbox{\hyperlink{classoomph_1_1BiCGStab}{Bi\+CGStab}}) may be preconditioned using\+: \begin{DoxyVerb}    BoomerAMG, Euclid or ParaSails
\end{DoxyVerb}
 By default \mbox{\hyperlink{classoomph_1_1GMRES}{GMRES}} without preconditioning is used.
