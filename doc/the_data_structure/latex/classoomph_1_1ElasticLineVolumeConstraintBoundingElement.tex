\doxysubsection{Detailed Description}
\subsubsection*{template$<$class ELEMENT$>$\newline
class oomph\+::\+Elastic\+Line\+Volume\+Constraint\+Bounding\+Element$<$ ELEMENT $>$}
/////////////////////////////////////////////////////////////////// /////////////////////////////////////////////////////////////////// 

The one-\/dimensional interface elements that allow imposition of a volume constraint specialised for the case when the nodal positions of the bulk elements are treated as solid degrees of freedom. To enforce that a fluid volume has a certain volume, attach these elements to all faces of the (2D cartesian) bulk fluid elements (of type ELEMENT) that bound that region and then specify the \char`\"{}pressure\char`\"{} value that is traded for the constraint.
